## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

The development server will start at `http://localhost:8001`

### Build

```sh
# Build for production
npm run build

# Preview production build
npm run preview
```
